#  JointAnglePrediction.00_matlab_to_python

*** You must run this script in the folder with the dataset ***
*** The resulting h5 file is used in the framework ***

Pathway to run matlab to python - 
    running_check_uniques.m / walking_check_uniques.m
    running_mat_to_h5.m / walking_mat_to_h5.m
    
Files - 
    running_check_uniques.m / walking_check_uniques.m - Confirms whether subjects with same ID are unique
    running_mat_to_h5.m / walking_mat_to_h5.m - Converts .mat file to .h5 file, which is then used in python processing
